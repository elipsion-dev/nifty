How to play

Press Start game and watch the pads light up in order, each with its own tone. When the sequence finishes, repeat it by typing the same keys on your keyboard. Get it right and the next round adds one more key; get one wrong and the round ends with your best chain length as your score.

What it trains

Fast typing is really memory in your fingers: you recall where a key is and press it without thinking. Key Memory exercises that recall directly. Holding a growing chain of key positions in mind — and then reproducing it by feel — is a playful workout for the same spatial memory that touch typing depends on. The tones give each key an extra hook, which makes the positions easier to remember.

Tips for a longer chain

Do not rush the playback — watch the whole sequence before you start repeating it. Many players find it helps to "hear" the little melody the tones make, or to picture the shape the keys trace on the keyboard. Keep your hands resting on the keys so your fingers can find each position quickly, and breathe — memory works better when you are relaxed.

Frequently asked questions

Is it really free? Yes. No sign-up, no email, no download, and no paywall.

Do you store my scores? No. The game runs entirely in your browser and nothing is uploaded. There is no account and no server keeping a record.

How does this help my typing? It strengthens the spatial memory for key positions that lets you type without looking — the heart of touch typing.

Does it work on a phone? It is designed for a physical keyboard. The memory challenge works best when you can feel the keys.

Important

Key Memory is a free typing game for practice and fun. It is a learning aid, not a formal typing certification, and nothing is saved or sent anywhere.
